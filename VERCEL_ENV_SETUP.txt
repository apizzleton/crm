Vercel Environment Variables Setup
==================================

Copy these into Vercel Dashboard → Settings → Environment Variables:

DATABASE_URL=postgresql://postgres.sbuambrtlkxxezszangx:c7OHGeA8O1v6HZ5u@aws-1-us-east-1.pooler.supabase.com:5432/postgres?sslmode=require

SECRET_KEY=<generate a random string using: openssl rand -hex 32>

Instructions:
1. Go to https://vercel.com/dashboard
2. Select your project
3. Go to Settings → Environment Variables
4. Add DATABASE_URL (value above)
5. Generate SECRET_KEY: Run `openssl rand -hex 32` in terminal, copy the output
6. Add SECRET_KEY with the generated value
7. Make sure to select all environments (Production, Preview, Development)
8. Click "Save"
